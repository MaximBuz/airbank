### Run locally

<b>Install all dependencies:</b>
```bash
npm install
```
```bash
cd frontend/
npm install
```
```bash
cd backend/
npm install
```
<b>Add database connection:</b>
```bash
cd backend/
touch .env
```
* add env-variable <code>DATABASE_URL="postgresql://DB_USER:DB_PASSWORD@HOST:PORT/airbank?schema=public"</code>

<b>Run from root folder:</b>
```bash
npm run start:dev
```
